Dataleap gives you two main ways to understand what your agents are doing.

## Usage page

Open **Usage** from the sidebar to review:

- Your weekly credit limit
- Usage trends over time
- Which agents are using credits

Use this view when you want a workspace-level picture of activity.

## Runs inside an agent

Open an agent and then open **Runs** to inspect individual executions.

The runs panel shows:

- Current runs
- Previous runs
- Execution status
- Output from the run
- Duration and credit usage

## Inspect a run

<Steps>
<Step title="Open the agent">

Go to **Agents** and choose the agent you want to inspect.

</Step>
<Step title="Open Runs">

In the side panel, click **Runs**.

</Step>
<Step title="Select a run">

Choose a current or previous run to open the full run view.

</Step>
<Step title="Review status and output">

Use the run details page to check whether the run succeeded, how long it took, and what it produced.

</Step>
</Steps>
