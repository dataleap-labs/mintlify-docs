---
title: "Add knowledge"
description: "Upload reference files so an agent can use your internal documents while it works."
---

Use the **Knowledge** panel when you want an agent to work from documents you provide.

## Upload files

<Steps>
<Step title="Open the agent">

Go to **Agents** and open the agent you want to update.

</Step>
<Step title="Open Knowledge">

In the side panel, click **Knowledge**.

</Step>
<Step title="Upload your files">

Drag files into the uploader or click **Browse files**.

</Step>
<Step title="Wait for processing to finish">

After the upload completes, the document will appear in the list for that agent.

</Step>
</Steps>

## Supported files

- `PDF`
- `DOCX`
- `PPT`
- `PPTX`
- `TXT`

Each file can be up to `10 MB`.

## Manage uploaded documents

From the document list, you can:

- Review the files attached to the agent.
- Download a file.
- Delete a file that is outdated or no longer relevant.

## When to use knowledge

Use knowledge for stable reference material such as:

- Playbooks
- Policies
- Product notes
- Reference documents the agent should consult repeatedly

<Warning>
  Keep the knowledge base current. Outdated files can lead to outdated answers or actions.
</Warning>
