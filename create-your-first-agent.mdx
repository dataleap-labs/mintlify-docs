The fastest way to get started is to describe the work you want done.

## Build from a prompt

<Steps>
<Step title="Open New Agent">

In the sidebar, click **New Agent**.

</Step>
<Step title="Describe the job clearly">

Write what you want the agent to do, the outcome you expect, and any important context.

If you have supporting files, attach them before you submit.

</Step>
<Step title="Click Build">

Dataleap will create an agent draft from your request and open it in the agent view.

</Step>
<Step title="Refine the draft in chat">

Ask follow-up questions, tighten the instructions, or tell the agent what to change.

</Step>
</Steps>

## Write better first prompts

- Start with one job, not a list of unrelated tasks.
- Include the source of truth the agent should use.
- State the final output you want.
- Add examples if the format matters.

## After the agent is created

Once the agent opens, you can keep improving it:

- Use **Workflow** to review instructions, triggers, and execution settings.
- Use **Connected Apps** to add access to external tools.
- Use **Knowledge** to upload reference documents when that panel is available.
- Use chat to test the agent and ask for changes.

<Tip>
  If you are not sure how to word the task, start from a template and then customize it.
</Tip>
