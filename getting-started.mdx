Use this page to get familiar with the app before you start building.

<Info>
  You need a Dataleap account and access to `https://app.dataleap.ai`.
</Info>

## First-time setup

<Steps>
<Step title="Sign in to Dataleap">

Open `https://app.dataleap.ai` and sign in.

</Step>
<Step title="Complete your profile if prompted">

Some workspaces ask you to add your first and last name before you continue.

</Step>
<Step title="Open the main workspace">

After you sign in, you will land in the main app where you can create agents, browse templates, and review usage.

</Step>
</Steps>

## Know the main navigation

- **New Agent** starts a new agent from a prompt.
- **Agents** shows the agents you already created.
- **Templates** shows example workflows you can reuse.
- **Usage** shows your credit usage and agent activity.
- **Settings** lets you manage your account and connected apps.

## What to do next

If you already know the task you want to automate, start with `/create-your-first-agent`.

If you want ideas first, start with `/use-templates`.
